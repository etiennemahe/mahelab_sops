3. Vortex standards and specimens well; ensure specimens are fully thawed prior to use 4. For standards, combine 190 uL working solution and 10 uL specimen 5. For query specimens, combine 195 uL working solution with 5 uL specimen 6. Vortex each tube x 5 seconds; incubate the tubes at room temperature x 2 minutes 7. Tap the screen gently to wake the Qubit; select the appropriate specimen & assay type 8. Read each query specimen one at a time, ensuring that specimen volume is set accordingly (ie at 5 uL); record the concentrations (in ng/uL) in the specimen database NOTE: Any leftover working solution should be stored in the 4C fridge
